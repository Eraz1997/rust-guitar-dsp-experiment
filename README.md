# Rust Experiments

## Setup

### Common

1. [Install Rust](https://www.rust-lang.org/tools/install)
2. [Install `rustfmt`](https://github.com/rust-lang/rustfmt)
3. [Install `clippy`](https://github.com/rust-lang/rust-clippy)

### Linux

```sh
sudo apt update
sudo apt install libjack-jackd2-dev
```

### Windows and Mac

You're all set!
